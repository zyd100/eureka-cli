# 基于eureka的微服务快速开发脚手架

# 初期搭建

| 功能           | 框架实现           |
| -------------- | ------------------ |
| 注册中心       | Eureka             |
| 配置中心       | SpringCloudConfig  |
| 负载均衡       | Ribbon             |
| 断路器         | Hystrix            |
| Rest服务式调用 | Feign(okhttp)      |
| 网关           | SpringCloudGateWay |
| 接口文档       | Swagger            |
| 数据库         | MySql              |
| ORM            | MyBatisPlus        |

# 二阶段目标
